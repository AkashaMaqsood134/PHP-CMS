# PHP-CMS

PHP-CMS is a **Content Management System** implemented in PHP, designed to manage users, roles, permissions, countries, cities, and states from a simple admin panel. It is built using **Bootstrap 4** and the **SB Admin 2 template** for a responsive and modern interface.

---

## Features

- **User Management**
  - Add, edit, delete users
  - Assign roles and manage status

- **Role Management**
  - Add, edit, delete roles
  - Manage role status

- **Permission Management**
  - Add, edit, delete permissions
  - Assign permissions to roles

- **Geographical Management**
  - Manage countries, cities, and states

- **Admin Dashboard**
  - Responsive layout using SB Admin 2
  - Data tables with sorting and pagination
  - Topbar notifications and messages

---

## Technologies Used

- PHP
- MySQL
- Bootstrap 4
- SB Admin 2 Template
- jQuery
- DataTables

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AkashaMaqsood134/PHP-CMS.git
2. Copy the project folder to your web server root (e.g., htdocs for XAMPP).

3. Create a MySQL database and import the db.sql file (if provided) or create the necessary tables (users, roles, permissions, etc.).

4. Update the database connection in include/db.php with your credentials.

5. Open the project in your browser:
 http://localhost/PHP-CMS

## Usage
Navigate to Dashboard to view overall statistics.

Go to Tables to manage users, roles, permissions, and locations.

Use the Add, Edit, and Delete buttons to perform CRUD operations.
